# temperature_
## 温度データをパブリッシュするROS 2パッケージ

## 概要
`mypkg`は、ランダムな温度データをROS 2トピック `/temperature` にパブリッシュするノードを提供します。他のノードがこのデータを購読し、アプリケーション内で利用できます。

## ノード
### `temperature`
- **役割:** ランダムな温度データを1秒ごとにパブリッシュします。
- **トピック:** `/temperature` (型: `std_msgs/Float32`)
- **機能:**
  - 温度データ (摂氏) を生成。
  - 他のノードが利用可能な形で公開。

## トピック
- `/temperature` (型: `std_msgs/Float32`)
  - **内容:** 温度データ (例: `22.3`)

## 使用方法
パッケージをビルド後、以下のコマンドで実行してください。

```bash
ros2 run mypkg temperature
